1. 손목관절 구조 (Radiocarpal Joint)

관절 형태 및 구성

항목내용
관절 형태타원관절 (Ellipsoidal Joint)
근위 구성요골(Radius) 원위부 + 척골 관절원판(Articular Disc)
원위 구성근위 수근골 열 — 주상골, 월상골, 삼각골

수근골 8개 (Carpal Bones)

근위열 (Proximal Row)

  • 주상골 (Scaphoid) — 가장 많이 골절
  • 월상골 (Lunate) — 가장 많이 탈구
  • 삼각골 (Triquetrum)
  • 두상골 (Pisiform) — 기능성 종자골

원위열 (Distal Row)

  • 대능형골 (Trapezium)
  • 소능형골 (Trapezoid)
  • 유두골 (Capitate) — 가장 큰 수근골
  • 유구골 (Hamate) — 갈고리 돌기 보유
암기법: "So Long To Pinky, The Carpals Here"
Scaphoid – Lunate – Triquetrum – Pisiform / Trapezium – Trapezoid – Capitate – Hamate

2. 손목관절 운동 (Kinematics)

운동 방향정상 ROM
굴곡 (Flexion)0° ~ 80°
신전 (Extension)0° ~ 70°
요측 편위 (Radial Deviation)0° ~ 20°
척측 편위 (Ulnar Deviation)0° ~ 30°

수근골 운동 메커니즘

굴곡 시

근위열 수근골 → 배측(dorsal)으로 회전

신전 시

근위열 수근골 → 장측(palmar)으로 회전

요측 편위 시

주상골 굴곡 → 근위열 굴곡
DISI 패턴

척측 편위 시

주상골 신전 → 근위열 신전
VISI 패턴

주요 근육

근육동작신경 지배
요측수근굴근 (FCR)굴곡 + 요측편위정중신경 C6, C7
척측수근굴근 (FCU)굴곡 + 척측편위척골신경 C7, C8
장장근 (Palmaris Longus)굴곡정중신경 C7, C8
요측수근신근 장/단 (ECRL/ECRB)신전 + 요측편위요골신경 C6, C7
척측수근신근 (ECU)신전 + 척측편위요골신경 후골간신경 C7, C8

3. 손목 인대 (Ligaments)

외인성 인대 (Extrinsic)

인대위치기능
요수근 장측 인대장측굴곡 제한, 안정성
척수근 장측 인대장측안정성
요수근 배측 인대배측신전 제한

내인성 인대 (Intrinsic) — 임상 핵심

주상월상 인대 (Scapholunate Lig.)

손상 시 → DISI 변형

Watson Test 양성

월상삼각 인대 (Lunotriquetral Lig.)

손상 시 → VISI 변형

삼각섬유연골복합체 (TFCC)

구성: 관절원판 + 척측 측부 인대 + 배측/장측 요척 인대 + 척측 수근 신근건 건초
기능: 요척관절(DRUJ) 안정화, 척측 수근 하중 전달 (약 20%)
손상 기전: 넘어질 때 손 짚기, 전완 비틀림

4. 손목 주요 손상

① 주상골 골절 (Scaphoid Fracture)

발생 기전FOOSH (넘어지면서 손 짚기)
핵심 징후해부학적 비연(Anatomical Snuffbox) 압통
위험성혈액 공급 원위→근위 방향 → 근위부 골절 시 무혈성 괴사(AVN)
진단초기 X-ray 놓침 多 → MRI 또는 CT 필요
치료비전위성: 석고 고정 / 전위성: 수술

② 월상골 탈구 (Lunate Dislocation)

발생 기전강한 손목 과신전 + 척측 편위
증상손목 장측 돌출, 정중신경 압박 → 손가락 저린감
분류Mayfield Stage I~IV (I: 주상월상 파열 → IV: 월상골 완전 탈구)

③ TFCC 손상

증상척측 손목 통증, 파악력 약화, 전완 회전 시 통증
특수 검사Piano Key Sign, 척측 스트레스 검사
분류Palmer Type I (외상성) / Type II (퇴행성)

④ 드퀘르뱅 건초염 (De Quervain's Tenosynovitis)

이환 건장무지외전근(APL) + 단무지신근(EPB) — 제1구획
증상요측 손목·엄지 측 통증 및 압통
특수 검사Finkelstein Test — 엄지 주먹 + 척측 편위 시 요측 통증
치료스플린트, 스테로이드 주사, 필요 시 수술적 감압

⑤ 수근관 증후군 (CTS)

원인횡수근 인대 아래 정중신경 압박
증상엄지~3번째 손가락 저린감, 야간 악화, 무지구근 위축
특수 검사Phalen's Test (손목 최대 굴곡 60초), Tinel's Sign (수근관 타진)
치료야간 스플린트, 스테로이드 주사, 수술적 감압

5. 손관절 구조 (Hand Joints)

관절명약어형태주요 운동
손목손허리관절CMC안장관절(엄지) / 평면관절(2~5)대립(엄지)
손허리손가락관절MCP타원관절굴곡·신전·외전·내전
근위지절간관절PIP경첩관절굴곡 0~110°
원위지절간관절DIP경첩관절굴곡 0~90°

MCP 관절 측부 인대 — 임상 핵심

굴곡 시 팽팽(Taut) → 부목 고정은 반드시 굴곡 위치에서
신전 시 이완(Lax) → 신전 고정 시 유착·변형 위험

엄지 CMC 관절 (안장관절)

  • 가장 자유도가 높은 CMC 관절
  • 대립(Opposition) = 굴곡 + 외전 + 내회전
  • 핵심 인대: 전방 사관인대(Beak Ligament) → 손상 시 Bennett 골절

6. 손 주요 변형 및 손상

추지 변형 (Mallet Finger)

  • 원인: DIP 신건 파열 or 말절골 골절
  • 기전: DIP 굴곡 강제
  • 변형: DIP 굴곡 고정
  • 치료: DIP 신전 스플린트 6~8주

부토니에르 변형 (Boutonniere)

  • 원인: PIP 중심건(Central Slip) 파열
  • 변형: PIP 굴곡 + DIP 과신전
  • 기전: 측속이 장측 이동

백조목 변형 (Swan-neck)

  • 원인: 내재근 단축, 류마티스, 장측판 손상
  • 변형: PIP 과신전 + DIP 굴곡
  • 부토니에르의 반대 패턴

베네트 골절 (Bennett's Fx)

  • 위치: 제1 중수골 기저부 관절 내
  • 기전: 엄지 축 방향 압박
  • 특징: 장무지외전근이 큰 편을 당김
  • 치료: K-wire 고정

롤란도 골절 (Rolando's Fx)

  • 제1 중수골 기저부 분쇄 골절
  • T자/Y자형 골절선
  • 베네트보다 예후 나쁨

스키어 엄지 (Skier's Thumb)

  • MCP 척측 측부 인대 파열
  • 기전: 엄지 강제 외전
  • 검사: 척측 스트레스 검사
  • 치료: 스플린트 or 수술

9. 재활 원칙 (Rehabilitation)

급성기 (0~3일)

  • PRICE 원칙
  • 부종 조절
  • 통증 완화
  • 적절한 고정

아급성기 (3일~3주)

  • ROM 운동 시작
  • 부드러운 근력 운동
  • 부종 관리 지속

재활기 (3주~)

  • 점진적 저항 운동
  • 고유감각 훈련
  • 기능적 운동 패턴

복귀기

  • 스포츠 특이적 훈련
  • 테이핑/보호대
  • 재손상 예방 교육
