# 상지 운동손상과 실무실습 - 손목관절 & 손관절

---

## 1. 손목관절 (Wrist Joint / Radiocarpal Joint)

### 1-1. 관절 구조 (Anatomy)

| 구성 요소 | 내용 |
|-----------|------|
| 관절 형태 | 타원관절(Ellipsoidal Joint) |
| 근위 구성 | 요골(Radius) 원위부 + 척골(Ulna)의 관절원판(Articular Disc) |
| 원위 구성 | 근위 수근골 열: 주상골(Scaphoid), 월상골(Lunate), 삼각골(Triquetrum) |
| 두상골(Pisiform) | 삼각골 전면에 위치, 기능성 종자골 |

#### 수근골 (Carpal Bones) - 8개

**근위열 (Proximal Row)**
1. 주상골 (Scaphoid) - 가장 많이 골절되는 수근골
2. 월상골 (Lunate) - 가장 많이 탈구되는 수근골
3. 삼각골 (Triquetrum)
4. 두상골 (Pisiform)

**원위열 (Distal Row)**
5. 대능형골 (Trapezium)
6. 소능형골 (Trapezoid)
7. 유두골 (Capitate) - 수근골 중 가장 큼
8. 유구골 (Hamate) - 갈고리 돌기(Hook) 보유

> 암기법: **"So Long To Pinky, The Carpals Here"**
> Scaphoid - Lunate - Triquetrum - Pisiform / Trapezium - Trapezoid - Capitate - Hamate

---

### 1-2. 관절 운동 (Kinematics)

| 운동 방향 | 범위(ROM) |
|-----------|-----------|
| 굴곡 (Flexion) | 0° ~ 80° |
| 신전 (Extension) | 0° ~ 70° |
| 요측 편위 (Radial Deviation) | 0° ~ 20° |
| 척측 편위 (Ulnar Deviation) | 0° ~ 30° |
| 원주운동 (Circumduction) | 위 복합 |

#### 수근골 운동 메커니즘
- **굴곡 시**: 근위열 수근골이 배측(dorsal)으로 회전
- **신전 시**: 근위열 수근골이 장측(palmar)으로 회전
- **요측 편위 시**: 주상골 굴곡 → 근위열 전체 굴곡 (DISI 패턴)
- **척측 편위 시**: 주상골 신전 → 근위열 전체 신전 (VISI 패턴)

---

### 1-3. 인대 (Ligaments)

#### 외인성 인대 (Extrinsic Ligaments)
| 인대 | 위치 | 기능 |
|------|------|------|
| 요수근 장측 인대 (Palmar Radiocarpal Lig.) | 장측 | 굴곡 제한, 안정성 |
| 척수근 장측 인대 (Palmar Ulnocarpal Lig.) | 장측 | 안정성 |
| 요수근 배측 인대 (Dorsal Radiocarpal Lig.) | 배측 | 신전 제한 |

#### 내인성 인대 (Intrinsic Ligaments)
| 인대 | 중요성 |
|------|--------|
| 주상월상 인대 (Scapholunate Ligament) | 임상적으로 매우 중요, 손상 시 DISI 변형 |
| 월상삼각 인대 (Lunotriquetral Ligament) | 손상 시 VISI 변형 |

#### 삼각섬유연골복합체 (TFCC: Triangular Fibrocartilage Complex)
- 구성: 관절원판 + 척측 측부 인대 + 배측/장측 요척 인대 + 척측 수근 신근건 건초
- 기능: 요척관절(DRUJ) 안정화, 척측 수근 하중 전달 (약 20%)
- 손상: 넘어질 때 손 짚기, 척측 손목 통증의 주요 원인

---

### 1-4. 주요 근육 및 건 (Muscles & Tendons)

#### 굴곡근
| 근육 | 신경 지배 |
|------|-----------|
| 요측수근굴근 (FCR) | 정중신경 (C6, C7) |
| 척측수근굴근 (FCU) | 척골신경 (C7, C8) |
| 장장근 (Palmaris Longus) | 정중신경 (C7, C8) |

#### 신전근
| 근육 | 신경 지배 |
|------|-----------|
| 요측수근신근 장/단 (ECRL/ECRB) | 요골신경 (C6, C7) |
| 척측수근신근 (ECU) | 요골신경 후골간신경 (C7, C8) |

### 1-5. 손목 주요 손상

#### (1) 주상골 골절 (Scaphoid Fracture)
- 발생 기전: 넘어지면서 손 짚기 (FOOSH: Fall On Outstretched Hand)
- 특징: **해부학적 비연 (Anatomical Snuffbox) 압통** - 진단의 핵심
- 문제점: 혈액 공급이 원위부에서 근위부로 → 근위부 골절 시 **무혈성 괴사(AVN)** 위험
- 진단: 초기 X-ray에서 놓치는 경우 많음 → MRI 또는 CT 필요
- 치료: 비전위성 → 석고 고정, 전위성 → 수술

#### (2) 월상골 탈구 (Lunate Dislocation)
- 발생 기전: 강한 손목 과신전 + 척측 편위
- 증상: 손목 장측 돌출, 정중신경 압박 → 손가락 저린감
- Mayfield 분류 (Stage I → IV)
  - I: 주상월상 인대 파열
  - II: + 요유두 인대 파열 (수근골 중간 관절 불안정)
  - III: + 월상삼각 인대 파열 → 주변 월상골 탈구
  - IV: 월상골 완전 탈구

#### (3) 손목 삼각섬유연골복합체 손상 (TFCC Injury)
- 증상: 척측 손목 통증, 파악력 약화, 전완 회전 시 통증
- 검사: **피아노 건반 징후 (Piano Key Sign)**, **척측 스트레스 검사**
- Palmer 분류: Type I (외상성) / Type II (퇴행성)

#### (4) 드퀘르뱅 건초염 (De Quervain's Tenosynovitis)
- 이환 건: 장무지외전근(APL) + 단무지신근(EPB) - 1구획
- 증상: 요측 손목 통증, 엄지 측 압통
- 검사: **핀켈슈타인 검사 (Finkelstein Test)** 양성
- 치료: 보존치료(스플린트, 주사), 필요 시 수술적 감압

#### (5) 손목 건초염 / 결절종 (Ganglion Cyst)
- 위치: 배측 손목 가장 흔함 (주상월상 인대 위)
- 진단: 투과조명 검사 (Transillumination)

## 2. 손관절 (Hand Joints)

### 2-1. 관절 구조 분류

| 관절명 | 약어 | 구성 |
|--------|------|------|
| 손목손허리관절 | CMC | 수근골 + 중수골 |
| 손허리손가락관절 | MCP | 중수골 + 근위지골 |
| 근위지절간관절 | PIP | 근위지골 + 중간지골 |
| 원위지절간관절 | DIP | 중간지골 + 말절골 |

---

### 2-2. 엄지 CMC 관절 (Thumb Carpometacarpal Joint)

- 형태: **안장관절 (Saddle Joint)** → 가장 자유도가 높은 CMC 관절
- 운동:
  - 굴곡/신전 (수장면에서)
  - 외전/내전 (시상면에서)
  - 대립(Opposition) = 굴곡 + 외전 + 내회전
- 중요 인대: 전방 사관인대(Anterior Oblique Lig.) = Beak Ligament → 손상 시 Bennett 골절

---

### 2-3. MCP 관절 (Metacarpophalangeal Joint)

- 형태: **타원관절** (2~5번 손가락), 안장관절 (엄지)
- 운동: 굴곡/신전, 외전/내전
- 측부 인대 (Collateral Ligaments):
  - 굴곡 시 팽팽 (Taut) → 굴곡 위치에서 부목 고정 (부종/유착 예방)
  - 신전 시 이완 (Lax)
- 장측판 (Volar Plate): 과신전 방지

---

### 2-4. PIP 관절 (Proximal Interphalangeal Joint)

- 형태: **경첩관절 (Hinge Joint)**
- 굴곡 0° ~ 110°, 신전 0°
- **임상적으로 매우 중요한 관절**
- 안정화 구조: 측부 인대 + 장측판 + 신건

#### 주요 손상
| 손상 | 기전 | 변형 |
|------|------|------|
| 장측판 손상 | 과신전 | 과신전 변형, 백조목 변형 위험 |
| 중심건(Central Slip) 손상 | 굴곡 강제 + 저항 | 부토니에르 변형 (Boutonniere) |
| 측부 인대 손상 | 측방 충격 | 관절 불안정성 |

---

### 2-5. 손 관련 주요 변형

#### (1) 추지 변형 (Mallet Finger)
- 원인: 말절 신건 파열 또는 말절골 기시부 골절
- 기전: DIP 관절에 굴곡 강제
- 변형: **DIP 굴곡 고정** (신전 불가)
- 치료: DIP 신전 위치 스플린트 고정 6~8주

#### (2) 부토니에르 변형 (Boutonniere Deformity)
- 원인: PIP 관절 중심건(Central Slip) 파열
- 변형: **PIP 굴곡 + DIP 과신전**
- 기전: 측속(Lateral Band)이 장측으로 이동 → PIP 굴곡 유지, DIP 과신전

#### (3) 백조목 변형 (Swan-neck Deformity)
- 원인: 내재근 단축, 류마티스, 장측판 손상
- 변형: **PIP 과신전 + DIP 굴곡**
- 부토니에르의 반대 패턴

#### (4) 베네트 골절 (Bennett's Fracture)
- 위치: 제1 중수골 기저부 관절 내 골절
- 기전: 엄지 축 방향 압박 (주먹 쥔 채 충격)
- 특징: 전방 사관인대에 의해 소편이 고정, 큰 편은 장무지외전근에 의해 당김
- 치료: 정복 후 핀 고정 (K-wire)

#### (5) 롤란도 골절 (Rolando's Fracture)
- 위치: 제1 중수골 기저부 관절 내 **분쇄 골절** (T자/Y자형)
- 베네트보다 예후 나쁨

---

### 2-6. 손의 신경 (Nerve Innervation)

| 신경 | 지배 영역 | 손상 시 변형 |
|------|-----------|-------------|
| 정중신경 (Median N.) | 엄지~3번째 손가락 반까지 장측, 무지구근 | 원숭이 손 (Ape Hand), 맹세 손 |
| 척골신경 (Ulnar N.) | 4~5번째 손가락, 소지구근, 골간근 | 갈퀴 손 (Claw Hand) |
| 요골신경 (Radial N.) | 손 배측 요측 | 수하수 (Wrist Drop) |

#### 정중신경 손상 레벨별 증상
- **팔꿈치 위 손상**: 전완 회내 약화, 손목/손가락 굴곡 약화 + 감각 이상
- **손목 수준 손상**: 무지구근 약화 → 엄지 대립 불가 (원숭이 손)
- **수근관 증후군 (CTS)**: 손목 횡인대 아래 정중신경 압박

#### 수근관 증후군 (Carpal Tunnel Syndrome)
- 증상: 엄지~3번째 손가락 저린감, 야간 악화, Thenar 위축
- 검사:
  - **팔렌 검사 (Phalen's Test)**: 손목 최대 굴곡 60초 → 증상 유발
  - **티넬 징후 (Tinel's Sign)**: 수근관 타진 → 방사통
- 치료: 야간 스플린트, 스테로이드 주사, 수술적 감압

## 3. 손목·손 관절 평가 (Clinical Assessment)

### 3-1. 시진
- 부종, 변형, 근위축, 피부 색 변화 확인
- 휴식 자세에서 손가락 배열 (정상: 요측으로 수렴)

### 3-2. 촉진 (압통 위치)

| 압통 위치 | 의심 손상 |
|-----------|-----------|
| 해부학적 비연 (Anatomical Snuffbox) | 주상골 골절 |
| TFCC (척측 손목) | TFCC 손상 |
| 1구획 (요측 손목) | 드퀘르뱅 건초염 |
| 수근관 중앙 | 수근관 증후군 |

### 3-3. 특수 검사 (Special Tests)

| 검사명 | 방법 | 양성 | 의심 |
|--------|------|------|------|
| Finkelstein Test | 엄지 주먹 쥐고 척측 편위 | 요측 통증 | 드퀘르뱅 |
| Phalen's Test | 손목 최대 굴곡 60초 | 손가락 저린감 | CTS |
| Tinel's Sign | 수근관 타진 | 방사통 | CTS |
| Piano Key Sign | 척골 두부 누르기 | 스프링 느낌, 통증 | TFCC/DRUJ |
| Scaphoid Shift Test (Watson) | 주상골 압박 후 요측 편위 | 클릭, 통증 | 주상월상 불안정성 |
| Valgus/Varus Stress Test | MCP/PIP 측방 스트레스 | 느슨함, 통증 | 측부 인대 손상 |

## 4. 재활 원칙 (Rehabilitation Principles)

### 단계별 재활
1. **급성기 (0~3일)**: PRICE 원칙, 부종 조절, 통증 완화
2. **아급성기 (3일~3주)**: ROM 운동, 부드러운 근력 운동
3. **재활기 (3주~)**: 점진적 저항 운동, 고유감각 훈련
4. **복귀기**: 스포츠 특이적 훈련, 테이핑/보호대

### 테이핑 및 보호
- **주상골 골절 의심**: 즉시 고정 후 이송
- **MCP 측부 인대 손상**: Buddy Taping (인접 손가락과 함께 테이핑)
- **추지 변형**: DIP 신전 스플린트
- **드퀘르뱅**: 엄지 포함 손목 스플린트

## 5. 핵심 요약 정리

| 구분 | 핵심 키워드 |
|------|------------|
| 가장 많이 골절되는 수근골 | 주상골 (Scaphoid) |
| 가장 많이 탈구되는 수근골 | 월상골 (Lunate) |
| FOOSH 손상 | 주상골 골절, 월상골 탈구, 콜레스 골절 |
| 해부학적 비연 압통 | 주상골 골절 |
| TFCC 손상 증상 | 척측 손목 통증, 전완 회전 시 통증 |
| 추지 변형 고정 위치 | DIP 신전 위치 |
| 부토니에르 vs 백조목 | PIP굴곡+DIP과신전 vs PIP과신전+DIP굴곡 |
| 갈퀴 손 | 척골신경 손상 |
| 원숭이 손 | 정중신경 손상 (무지구근 위축) |
| 수하수 | 요골신경 손상 |
| CTS 특수검사 | Phalen's Test, Tinel's Sign |
| 드퀘르뱅 특수검사 | Finkelstein Test |
